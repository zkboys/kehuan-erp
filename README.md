# react-demo

组织架构管理系统demo项目，基于node react redux webpack antd搭建

## 安装&启动
1. [node > 7.x](https://nodejs.org/en/)
2. [pm2](http://pm2.keymetrics.io/) 
```
yarn install

yarn add pm2@latest -g

npm run pro 

访问 http://localhost:3000
```

## 查看端口占用
```
lsof -i:3000
```
